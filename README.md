## The Game of Frogger

Game engine and Art Assets from Udacity Front End Course

### Basic Functionality

The goal of the game is for the player to reach the water, without colliding into any one of the enemies. The player can move left, right, up and down. The enemies move in varying speeds on the paved block portion of the scene. Once a the player collides with an enemy, the game is reset and the player moves back to the start square. Once the player reaches the water the game is won.

### Additional Functionality

* Player selection: allow the user to select the image for the player character before starting the game. 
* Score: The score increases each time the player reaches the water, and it resets to 0 when a collision occurs
